Version 1.2

March 30th, 2026 — platforms: AUv3 and Standalone for iPhone, iPad, Mac; VST3 for Mac and Windows

New Features

  • VST3 plugin format — now available for macOS and Windows alongside AUv3.
  • Built-in Tutorial — interactive, hands-on walkthrough of all drumming features. Opens automatically on first launch; start anytime from Settings → BUILT-IN TUTORIAL or by loading any preset from [Factory] Tutorial Presets in the Preset Browser.
  • Beat Monitor — animated dots next to knobs that flash in the instrument's color when a beat is generated or passes through the parameter, or gray when blocked. Pattern tab visualizes beats generated by Topographic/Euclidean engines and how Blend is mixing or blocking them, Linear tab visualizes priority and probability pass/block, Velocity tab visualizes blocked notes through velocity=0.
  • Pattern Tab — START ON beat monitor dot flashes once per Euclidean cycle on the START ON step only, showing the beginning of the Euclidean loop.
  • Master Sliders — toggleable panel replacing the XY Map with six master sliders: Density, Blend, Pre Probability, Post Probability, Level, and Chaos. Each slider has a Hard/Soft mode toggle. Hard mode overrides all per-channel values to the master value. Soft mode interpolates each channel independently from its current value. One Master Slider can be displayed under the XY Map, configurable in Settings. 🙏
  • Master Sliders — knobs controlled by Master Sliders now show Limit Ticks — two small marks on the knob arc indicating the effective range defined by the master slider. In Soft mode, ticks show where the master constrains the knob. In Hard mode, ticks spread to min/max showing the full range is available. Blend knobs show two moving ticks (floor and ceiling) reflecting the bipolar constraint. Limit Ticks can be hidden in Settings.
  • New Preset and MIDI Mapping Browser with 3-column folder browser and info panel, replacing the old dropdown menus. Supports inline Save As / New Folder / Rename editors, folder deletion, and Keep Open toggle for preset auditioning without closing. Includes new Author and Description metadata fields. 🙏
  • Settings — hold an LED and scroll the NOTE to audition each note as you scroll through (iPad/iPhone only).
  • Linear Tab — paint mode in priority matrix — tap and drag across cells to assign priorities in one gesture, with double-tap to select an entire column. 🙏
  • Velocity Tab — new 1N (whole note) knob added to the Velocity Bender, providing a slow one-bar velocity sweep cycle.

Improvements

  • New Settings window with three panels — Instruments (rename), Controls (toggles and dropdowns), and Options (Linear Grid). Accessible via gear icon button.
  • New NOTE Tab — MIDI note assignment, MIDI Mapping browser, NOTE LENGTH, MIDI CHANNEL, and standalone controls (MIDI OUT, BPM) consolidated into one tab.
  • NOTE Tab — MIDI CHANNEL (1–16) is now accessible in AUv3 and VST3 plugin mode — no longer locked to Channel 10. Default remains 10 (GM Drums).
  • Velocity Tab — RANDOM knob now crossfades between fixed LEVEL velocity and fully random (0–127), so RANDOM at 100% always produces the full velocity range regardless of LEVEL setting.
  • Velocity Tab — RANDOM/RESET button replaced with compact square icons (dice for randomize, line for reset).
  • Standalone — Play/Stop button moved from the chaos slider area to the tab bar, above the LED column.
  • Standalone — minimum window size now matches the GUI minimum, preventing layout overflow when resizing too small.
  • Scrollable lists now commit on release: dragging previews the value visually but the parameter only changes when you lift your finger, preventing jarring mid-playback changes. Affects STEPS, PULSES, START ON, CLOCK, NOTE, NOTE LENGTH, LINEAR GRID, BPM, and MIDI CHANNEL. 🙏
  • MIDI Mappings — mapping file presetVersion bumped from 1.0 to 2.0 to support new metadata fields.

Bug Fixes

  • Presets — preset names now persist correctly when saving and reopening a DAW project.
  • Presets — switching presets or loading a DAW project no longer leaves hanging MIDI notes.
  • Stopping transport no longer leaves hanging MIDI notes. 🙏
  • Pattern Tab — Euclidean engine no longer skips the first beat after transport stop/start.
  • AUv3 — macOS AUv3 plugin window is now resizable with aspect ratio lock, matching standalone behavior. 🙏

Version 1.1

March 5th, 2026 — platforms: AUv3 and Standalone for iPhone, iPad, Mac

Improvements

  • Presets and MIDI Mappings — No longer store redundant names inside XML files — filenames are the single source of truth.
  • Presets and MIDI Mappings — Menu dropdowns now scroll the entire list including subfolders, so all items are reachable even when there are many subfolder categories.
  • Pattern Tab — 10x higher-resolution of SWING, SHIFT, and HUMANIZE parameters with reduced knob sensitivity for precise control. 🙏
  • Pattern Tab — Added kinetic scrolling to STEPS, PULSES, START ON, and NOTE lists for smoother, more precise, and faster navigation.
  • Pattern Tab - BLEND header now shows arrows pointing toward the Grids and Euclidean knobs, indicating which set of controls becomes more influential as you move the BLEND position.
  • Velocity Tab — LEVEL knob now goes down to 0, allowing the channel to be fully muted (equivalent of MIDI message 0x9n "Note On velocity=0"). Previously the minimum value was 1.
  • Velocity Tab — Moved velocity LEVEL and RANDOM processing after LINEAR DRUMMING in the signal chain to match GUI tab order; no change in behaviour.
  • Settings Tab — A single MIDI Note selector (e.g., "36-C1", "42-F#1") replacing NOTE and OCTAVE columns — shows both MIDI number and note name for quicker selection and easier MIDI monitoring.
  • Settings Tab — Added 30+ MIDI Mappings for apps and hardware instruments. 🙏
  • Settings Tab — MIDI Mappings menu now shows an asterisk "*" indicator when any MIDI note assignment is changed.
  • Settings Tab — Tapping a different folder while a submenu is open now switches to it in one tap instead of two. 🙏
  • LED header text doesn't change when switching between tabs and only changes when you tap it.
  • Standalone Mode — The app now opens in landscape on iPhone for a better workflow.

Bug Fixes

  • Presets and Settings Tab — Fixed user Preset and MIDI Mapping saving on macOS; also replaced native file dialog with in-app Save As dialog on all platforms.
  • Settings Tab — Selecting a MIDI OUT device in the Settings tab now correctly picks the chosen device. 🙏

Version 1.0

February 23rd, 2026 — platforms: AUv3 and Standalone for iPhone, iPad, Standalone for Mac

New

  • AUv3 MIDI Processor — works in Logic Pro, GarageBand, AUM, Cubasis, BeatMaker 3, Drambo, Loopy Pro, Audiobus, and more.
  • 6 independent instrument channels: BD, SN, HH + 3 accent channels BD', SN', HH'.
  • Algorithmic rhythm generator based on Mutable Instruments Grids Eurorack module with per-channel density and chaos controls.
  • Euclidean rhythm generator (Bjorklund algorithm) with per-channel steps, pulses, and start offset.
  • Per-channel Blend control to mix between Grids and Euclidean engines.
  • XY pad for real-time navigation of the Grids pattern topology, with optional colored visual feedback on note triggers.
  • Per-channel clock ratios from /8 to ×8 for polyrhythmic patterns.
  • Per-channel swing (Roger Linn algorithm, −99 to +99).
  • Per-channel timing shift to push notes ahead of or behind the beat.
  • Per-channel humanize for subtle micro-timing variation.
  • Linear Drumming — 6-level priority system preventing simultaneous instrument hits.
  • Configurable Linear Drumming grid resolution (64n / 32n / 16n / 8n / 4n).
  • Pre- and post-Linear Drumming probability, mute, and solo per channel.
  • Per-channel base velocity and random velocity variation.
  • Velocity Bender — shape dynamic accents across a bar at four freely adjustable beat divisions (2n, 4n, 4nt, 8n).
  • Real-time Velocity Bender waveform display.
  • Random Velocity Bender generator.
  • Per-channel Velocity Bender on/off.
  • Random preset generator based on existing presets.
  • Configurable MIDI note out length (4n / 8n / 16n / 32n / 64n).
  • MIDI mapping system — note assignments persist independently of preset changes.
  • Clickable LED pads for manual note triggering.
  • DAW integration — full parameter automation for all controls.
  • Preset and MIDI mapping files accessible via the iOS Files app.
  • Customizable instrument names.
  • Resizable app and plugin window.
  • Standalone app: internal clock with BPM control (40–240).
  • Standalone app: virtual and physical MIDI output and channel selection.
  • 150+ factory presets, including Euclidean patterns inspired by world rhythms.
  • Universal Purchase for iPad, iPhone, and Mac.
