Häufig gestellte Fragen

über Augmatic GRE

Was ist das blaue Banner / die Berechtigungsabfrage nach der Installation?

Dies ist eine einmalige iOS-Berechtigungsanfrage, damit das AUv3-Plugin auf den Dokumentenordner der Standalone-App zugreifen kann. Tippen Sie auf die blaue Schaltfläche „Enable Files App Access“, navigieren Sie zum Ordner von Augmatic GRE unter „Auf meinem iPad“ und drücken Sie Open. Ohne diese Berechtigung sind im AUv3 gespeicherte Presets in der Files app nicht sichtbar. Weitere Details finden Sie unter File Storage.

Wie fängt man am besten mit Augmatic GRE an?

Machen Sie kleine, schrittweise Änderungen und hören Sie aufmerksam zu. Die App ist darauf ausgelegt, sich entwickelnde Patterns zu erzeugen — viele Regler gleichzeitig zu bewegen kann chaotisch klingen, während subtile Parameteranpassungen musikalischere Ergebnisse liefern.

Warum sind manche Kanäle stumm, obwohl die Einstellungen hoch sind?

Das liegt fast immer an einer BLEND-Wechselwirkung. Wenn BLEND beispielsweise ganz links steht (nur Grids) und die Grids-DENSITY für einen bestimmten Kanal ebenfalls ganz links steht, gibt dieser Kanal nichts aus — selbst wenn die Euclidean-PULSES hoch eingestellt sind, da Euclidean-Noten bei dieser BLEND-Position nicht verwendet werden.

Beide Engines können auch durch die Linear Drumming-Matrix stummgeschaltet werden — wenn ein Kanal niedrige Priorität hat (Punkt rechts) und andere Kanäle häufig spielen.

Ich bekomme keinen Ton heraus. Was mache ich falsch?

Augmatic GRE ist ein reines MIDI-Plugin — es erzeugt keinen Audio-Sound. Sie müssen:

  1. Sicherstellen, dass die ausgegebenen MIDI-Noten mit den Noten übereinstimmen, die Ihr Drum-Sampler oder Synthesizer erwartet. Sie können die ausgegebenen MIDI-Noten im Settings-Tab auswählen.
  2. Die eingebauten MIDI-Mappings verwenden oder ein benutzerdefiniertes Mapping für Ihr Instrument erstellen und speichern, damit Sie die Einrichtung nicht wiederholen müssen.
  3. Den MIDI-Ausgang jedes Kanals an den entsprechenden Sound in Ihrer Drum-Engine routen.
Wofür sind die Kanäle BD', SN' und HH'? Wie macht man sie nützlicher?

BD', SN' und HH' sind Akzent-Kanäle, die den Hauptkanälen BD, SN und HH entsprechen. Im ursprünglichen Grids-Design repräsentieren Akzent-Kanäle schwächere Backbeat-Elemente und werden absichtlich seltener ausgelöst. Um dieses Verhalten nachzubilden, stellen Sie die Akzent-DENSITY auf etwa 25%.

In Augmatic GRE können Sie die Akzent-Kanäle nach Belieben nutzen. Bringen Sie sie mit einer DENSITY über 25% in aggressiveres Territorium, oder nutzen Sie sie, um längere Akzent-Phrasen über mehrere Takte zu erzeugen, indem Sie CLOCK auf /2 oder /4 setzen.

Wenn Sie langsameren Akzent-Kanälen höhere Priorität in der Linear Drumming-Matrix geben, erzeugt GRE deutlich interessantere, sich entwickelnde Patterns anstatt der typischen sich wiederholenden 1-Takt-Loops.

Wie kann ich zum Testen viele Schläge auf einem Kanal erzwingen?

Drehen Sie BLEND ganz nach rechts (Euclidean) und stellen Sie die Euclidean-Parameter auf 16 STEPS, 16 PULSES, START ON 1. Dies gibt schnelle Sechzehntelnoten aus.

Wie funktioniert der BLEND-Regler?

BLEND überblendet die Wahrscheinlichkeit zwischen den beiden Engines:

  • Ganz links → nur die Grids-Engine (XY Map) erzeugt Noten.
  • Ganz rechts → nur die Euclidean-Engine erzeugt Noten.
  • Jede Position dazwischen → eine Wahrscheinlichkeitsmischung (z. B. 12-Uhr-Position = 50/50 Chance, dass eine Note von einer der beiden Engines kommt).

Wenn BLEND ganz links steht, haben Euclidean-PULSES-Einstellungen keine Wirkung — und umgekehrt.

Was sind Master Sliders und wann sollte man sie verwenden?

Master Sliders ermöglichen die Steuerung von Gruppen kanalspezifischer Regler (Density, Chaos, Blend, Prob Pre, Prob Post, Level) mit einer einzigen Geste. Schalten Sie das Panel mit der Schaltfläche neben dem XY Map ein oder aus.

Jeder Slider hat einen Hard/Soft-Modus: Hard überschreibt alle Kanäle auf denselben Wert; Soft skaliert jeden Kanal relativ zu seiner individuellen Einstellung. Verwenden Sie sie für Live-Performance-Sweeps, schnelle Mix-Anpassungen oder die gleichzeitige Automation mehrerer Kanäle.

Was sind die blinkenden Punkte neben Reglern und Listen?

Das sind Beat Monitor-Punkte — sie bieten visuelles Feedback zum MIDI-Signalfluss. Jeder Punkt blinkt in der Farbe des Instruments, wenn ein Beat diese Verarbeitungsstufe passiert, oder grau, wenn er blockiert wird. Sie erscheinen auf jedem Tab: Pattern, Linear und Velocity.

Beat Monitor kann in den Settings ein-/ausgeschaltet werden. Siehe Interface Overview im Handbuch für eine vollständige Erklärung jedes Punktes.

Kann man Fills erzeugen?

Ja. Der Master Chaos-Slider (unter dem XY-Pad) ist dafür konzipiert. Erhöhen Sie ihn, um Variationen und Fills einzuführen, und bringen Sie ihn dann wieder zurück. Da es sich um einen exponierten AUv3-Parameter handelt, kann er von einer DAW automatisiert werden.

Weitere Tipps für Fills:

  • Das Erhöhen des Master Chaos-Sliders erzeugt Fills, die kohärent mit dem Basis-Pattern sind (da sie daraus abgeleitet werden).
  • Nutzen Sie das Master Sliders-Panel, um Density oder Blend zusammen mit Chaos zu sweepen, für dramatischere Fills.
  • Ein sehr langsamer LFO auf dem Master Chaos-Slider kann eine sich entwickelnde Struktur auf Song-Ebene erzeugen.
  • Nehmen Sie den MIDI-Ausgang auf, während Sie den Slider manuell bewegen, um Fills an bestimmten Stellen einzufangen.
Was ist Linear Drumming?

Linear Drumming begrenzt, wie viele Schläge gleichzeitig auf einem Step erklingen können, wodurch der Groove menschlicher und bewusster wirkt. Anstatt dass BD, SN, HH und ihre Akzent-Kanäle alle gleichzeitig feuern, wählen Sie, welcher Kanal den Step gewinnt, indem Sie seinen Punkt in der Matrix nach links bewegen. Je weiter rechts ein Punkt steht, desto niedriger ist seine Priorität — Instrumente mit höherer Priorität unterdrücken ihn und er spielt nur, wenn kein anderer Kanal in diesem Moment spielt.

Kann Linear Drumming als Hi-Hat-Choke verwendet werden?

Nicht direkt. Linear Drumming ist kein klassisches Choke-Feature. Wenn ein Instrument auf eine höhere Priorität gesetzt ist, unterdrückt es alle Instrumente mit niedrigerer Priorität — nicht nur ein bestimmtes zugeordnetes Instrument.

Für korrektes Choke-Verhalten (z. B. offene Hi-Hat wird durch geschlossene Hi-Hat abgeschnitten) verwenden Sie die Choke-Funktion in Ihrem Drum-Sampler oder Synthesizer.

SWING scheint nichts zu bewirken. Warum?

Swing erzeugt ein Shuffle-/Groove-Gefühl, indem jede zweite Sechzehntelnote verzögert wird. Wenn Ihr Pattern keine Sechzehntelnoten enthält, werden Sie den Effekt nicht hören. Um es zu testen, drehen Sie BLEND ganz nach rechts (Euclidean) und stellen Sie 16 STEPS, 16 PULSES, START ON 1 ein, um ein schnelles Sechzehntelnoten-Pattern zu erhalten, das sofort auf SWING-Änderungen reagiert. Das Erhöhen von CHAOS kann die Grids-Engine auch dazu bringen, einige Sechzehntelnoten hinzuzufügen.

Wie funktioniert negativer SHIFT (Vorwärtsverschiebung) genau?

Shift ist eine Nachbearbeitungsverzögerung, die nach der Pattern-Erzeugung angewendet wird. Sowohl Vorwärts- als auch Rückwärtsverschiebung verwenden eine PPQ-Nachschlagetabelle, um Noten um präzise Beträge zu verzögern. Vorwärtsverschiebung (Werte unterhalb der Mitte) verwendet größere Verzögerungswerte, sodass Noten kurz vor dem nächsten Zyklus landen und den Effekt erzeugen, dem Beat voraus zu spielen. Dies ermöglicht Flam-artige Schläge: eine Vorschlagnote, die kurz vor dem Hauptschlag landet und dem Sound mehr Gewicht und Charakter verleiht.

Hinweis: Das Anpassen von SHIFT in Echtzeit oder durch Automation — besonders beim Überqueren des Mittelpunkts — kann unerwartete Ergebnisse erzeugen. Der Parameter ist dennoch für Automation verfügbar, da subtile Bewegungen gut funktionieren.

Wie benennt man ein Preset oder MIDI Mapping um?

Öffnen Sie den Preset Browser oder MIDI Mapping Browser, wählen Sie die Datei aus und tippen Sie auf die Schaltfläche Rename im Aktionspanel. Geben Sie den neuen Namen ein und bestätigen Sie.

Alternativ können Sie auf iOS die XML-Datei direkt in der Files app unter Auf meinem iPad / Auf meinem iPhone → Augmatic GRE umbenennen.

Gibt es eine Möglichkeit, einen Snap-Punkt zu speichern und nach dem Experimentieren dorthin zurückzukehren?

Nicht direkt. Wenn Sie von einem Pattern ausgehen und nach dem Ändern einiger Parameter dorthin zurückkehren möchten, speichern Sie es als Preset und rufen Sie es bei Bedarf ab. Zukünftige Versionen von Augmatic GRE werden dies für Live-Performance vereinfachen.

Unterstützen Regler Doppeltippen zum Zurücksetzen?

Ja, alle Regler unterstützen Doppeltippen zum Zurücksetzen auf den Werksstandard. Eine benutzerdefinierte Standardposition kann nicht festgelegt werden.

Das Init-Preset ist nicht leer, oder?

Richtig — das Init-Preset ist nicht leer. Es ist ein einfacher Four-on-the-Floor-Ausgangspunkt in beiden Engines, Grids und Euclidean. Von dort aus wird alles, was Sie tun, das Pattern nur interessanter machen.

Können alle Parameter automatisiert werden?

Ja. Alle Parameter sind als AUv3- und VST3-Parameter verfügbar und können von jeder kompatiblen Host-DAW automatisiert werden.

Gibt es kreative Automations-Ideen?
  • Sehr langsamer LFO auf Master Chaos → erzeugt automatisch eine sich entwickelnde Struktur auf Song-Ebene.
  • XY-Pad-Position automatisieren → morpht im Laufe der Zeit zwischen verschiedenen Drum-Pattern-Topologien.
  • Individuelle Density/Chaos automatisieren → Patterns auf- und abbauen für Strophe-/Refrain-Übergänge.
Kann ich die MIDI-Noten für jede der 6 Spuren mit einem MIDI-Sequencer oder einer externen Quelle randomisieren?

Ja! Alle Parameter in Augmatic GRE können automatisiert werden, einschließlich der MIDI-OUT-Noten für jede Spur (BD, SN, HH usw.). Das bedeutet, sie können auch Ziel für Randomisierung sein.

So richten Sie es in AUM ein:

  1. Tippen Sie auf das Hamburger-Menü links von Augmatic GRE und wählen Sie dann das Regler-Symbol (AU Parameters) aus dem Menü auf der rechten Seite.
  2. Sie sehen eine vollständige Liste der Parameter — die MIDI-Note-Parameter (BD MIDI Note, SN MIDI Note, HH MIDI Note usw.) befinden sich am Ende, Tab für Tab geordnet, dem GUI-Layout von links nach rechts folgend.
  3. Für jeden Note-Parameter können Sie eine CC-Nummer zuweisen. Wenn Sie ein Plugin haben, das Sample & Hold-Werte über CC sendet, wird dieses Signal die Note randomisieren — genau das, was Sie suchen.
Auf wie vielen Geräten kann ich Augmatic GRE installieren?

App Store-Version (AUv3): Installation auf beliebig vielen Geräten, die mit derselben Apple ID angemeldet sind — kein zusätzlicher Kauf erforderlich.

Desktop-Version (VST3): Als Einzelperson dürfen Sie die Software auf beliebig vielen Computern installieren und aktivieren, die Sie persönlich besitzen und kontrollieren, solange nur Sie sie nutzen. Geschäfts- und Organisationslizenzen gelten pro Gerät — für jeden Computer ist eine separate Lizenz erforderlich.
